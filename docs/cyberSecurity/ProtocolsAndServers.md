### 协议和服务器

1.常用协议的默认端口号


| 协议     | TCP端口 | 应用程序                        | 数据安全性 |
|---------|---------|---------------------------------|------------|
| FTP     | 21      | 文件传输                         | 明文       |
| FTPS    | 990     | 文件传输                         | 加密       |
| HTTP    | 80      | Worldwide Web                        | 明文       |
| HTTPS   | 443     | Worldwide Web                         | 加密       |
| IMAP    | 143     | 电子邮件 (MDA)                   | 明文       |
| IMAPS   | 993     | 电子邮件 (MDA)                   | 加密       |
| POP3    | 110     | 电子邮件 (MDA)                   | 明文       |
| POP3S   | 995     | 电子邮件 (MDA)                   | 加密       |
| SFTP    | 22      | 文件传输                         | 加密       |
| SSH     | 22      | 远程访问和文件传输               | 加密       |
| SMTP    | 25      | 电子邮件 (MTA)                   | 明文       |
| SMTPS   | 465     | 电子邮件 (MTA)                   | 加密       |
| Telnet  | 23      | 远程访问                         | 明文       |
